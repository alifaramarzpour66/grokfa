# دیباگ کد پایتون (لیست خالی)

### پرامپت:
### خروجی واقعی گروک:
> کد اصلی مشکلی نداره و برای لیست خالی عدد 0 برمی‌گردونه (درست).  
> ولی اگه می‌خوای صریح‌تر باشه:

```python
def sum_even(numbers):
    return sum(n for n in numbers if n % 2 == 0)

# تست
print(sum_even([]))        # → 0
print(sum_even([1,2,3,4])) # → 6
